① Chuẩn hóa ITN theo từ điển — normalize.py áp itn_rules pack lên transcript (sửa đúng lỗi ASR đã quan sát).
② Trigger spotting — rapidfuzz trên MỌI partial, cửa sổ đuôi + dominance filter (hết nhầm "cứu hộ ô tô" vs "xe máy"); final quét cả câu.
③ Extraction — Groq llama-3.3-70b JSON mode theo schema pack, mỗi field kèm confidence + evidence quote; retry 4 tầng (429/mạng/parse rỗng) → fallback 8b-instant.
④ NER local PyTorch đối chiếu độc lập (PERSON/LOCATION/MISC) → agreement.

🔥 PyTorch layer (local, CPU)

silero-VAD nghe stream PCM — người nói ngừng 0.8s → tự audio.commit cho RTT → final về sớm, form điền sớm.

NER tiếng Việt (NlpHUST electra) đối chiếu độc lập với LLM → tăng/giảm điểm + cờ "NER không xác nhận".

Thiếu deps → degrade sạch, pipeline vẫn chạy.

② Action Call — 2 chính sách

🚨 AUTO — khẩn cấp, bắn ngay khi nói

Nghe "bấm nút gửi yêu cầu cứu hộ xe máy" → nút ARMED sáng trong 0.3ms ngay trên partial → final xác nhận → tự fire: ticket + PDF + TTS "Đã ghi nhận, xe cứu hộ sẽ đến trong 20 phút".

🖐️ CLICK — hồ sơ/y lệnh, chờ người xác nhận

Armed sáng chờ giám định viên/bác sĩ bấm, hoặc đi qua Màn Duyệt & chấm điểm rồi GỬI — đúng chuẩn compliance ngành.
ISSUE_ELECTRONIC_PRESCRIPTION · ORDER_LAB_TESTS · SCHEDULE_FOLLOW_UP · SUBMIT_*_REPORT · SCHEDULE_SURVEYOR_VISIT…

③ Bên trong một Domain Pack — "một engine, mọi nghiệp vụ"

📋 form.sections[]

Schema field (text/list/enum/number + synonyms + item_hint) → UI render động và khuôn JSON cho LLM. Đổi pack = đổi form, 0 dòng code.

⚡ actions[]

id + label + triggers[] (2–4 biến thể) + policy auto|click + required_fields + câu TTS xác nhận + template PDF.

🗣️ hint_terms + specialties

Thuật ngữ code-switch phải giữ nguyên (EF, HP, LDL-C, tên thuốc, biển số mẫu) → build thành hint_text bơm vào VALSEA RTT. Y tế: 3 module chuyên khoa đúng phạm vi test.

🔧 itn_rules[]

Từ điển sửa ASR hậu kỳ: "mèn gần"→"men gan", IF 60%→EF 60%, alpha-chloine→Alpha Choay… — chính là chỗ doanh nghiệp "dạy" hệ thống ngôn ngữ ngành mình.

🎯 scoring.validators

Regex biển số VN, đơn vị liều… → FormScorer chấm + attention list. ASR nuốt số là máy tự thú chứ không im lặng sai.

📝 few_shots + instructions

1 ví dụ vàng + luật nghiệp vụ (phân biệt thuốc mới/duy trì/chỉnh liều; không lấy câu trấn an làm chẩn đoán; không suy diễn danh xưng).
